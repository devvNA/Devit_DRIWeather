## Instalasi

1. Clone repository

```bash
git clone https://github.com/username/devit_driweather.git
```

2. Install dependencies

```bash
flutter pub get
```

3. Tambahkan API key di file .env

```
API_KEY=your_tomorrow_io_api_key
```

4. Jalankan aplikasi

```bash
flutter run
```
